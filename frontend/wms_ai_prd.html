# WMS-AI Product Requirements Document ## Project Overview WMS-AI is an intelligent warehouse management system designed for Admin and Staff users. It streamlines inventory tracking, order management, and provides AI-driven insights for optimized warehouse operations. ## User Roles - **Admin**: Full access to all modules, system settings, and user management. - **Staff**: Access to operational modules (Products, Sales, Orders) with restricted access to administrative settings. ## Core Modules 1. **Authentication**: Login, Forgot Password, Reset Password. 2. **Dashboard**: KPI metrics, sales charts, inventory health, and AI recommendations. 3. **Inventory Management**: Products, Categories, Stock In/Out. 4. **Relationship Management**: Suppliers. 5. **Operations**: Purchase Orders, Sales Transactions. 6. **Intelligence**: Alerts Center, AI Recommendations, Advanced Analytics. 7. **Utilities**: File Import/Export. 8. **Administration**: User Management, System Settings. ## Visual Identity - **Tone**: Professional, Reliable, Enterprise-grade. - **Layout**: Collapsible left sidebar, sticky top header, responsive content area. - **Feedback**: Color-coded badges for statuses (Low Stock, Pending, etc.), toast notifications, and confirmation modals.
